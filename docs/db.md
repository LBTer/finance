销售记录：
# 类型1，美金订单
- 销售记录的字段包括：
  * 订单号（唯一标识）
  * 经办人（关联用户ID，创建记录的人）
  * 创建时间
  * 更新时间
  * 产品名称
  * 类别
  * 数量
  * 单价（美元）
  * 总价（美元）
  * 汇率（美元-人民币）
  * 运费（陆内）- 人民币
  * 运费（海运）- 人民币
  * 物流公司
  * 退款金额 - 人民币
  * 退税金额 - 人民币
  * 利润（最后计算填写的金额）- 人民币
  * 状态（待审核/已审核/已拒绝）
  * 备注
